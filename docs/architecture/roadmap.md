# Roadmap MVP

## Fase 1

- Estabilizar la base compilable de `shared`, `composeApp` y `server`.
- Definir navegacion minima y placeholders funcionales.
- Documentar arquitectura y decisiones iniciales.

## Fase 2

- Conectar login real y sesion.
- Anadir repositorios con backend HTTP.
- Implementar formularios reales de ticket y respuesta.

## Fase 3

- Persistencia real en server.
- Adjuntos reales.
- Filtros y busquedas mas completas.
- Panel admin mas util.

## Fase 4

- Notificaciones moviles reales.
- Mejoras de iOS.
- Hardening de seguridad y observabilidad.
