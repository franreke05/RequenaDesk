# Portfolio Case Study Outline

## Problema

Necesitaba una base profesional para gestionar tickets de clientes freelance desde una app desktop propia y un backend ligero.

## Objetivo

Construir un MVP limpio y escalable con Kotlin Multiplatform, Compose Multiplatform y Ktor.

## Enfoque tecnico

- Desktop como canal principal.
- Android lite para notificaciones y revision rapida.
- Shared module para dominio, contratos y viewmodels.
- Server Ktor con rutas base y estructura preparada para crecer.

## Que demuestra este proyecto

- Capacidad de disenar una arquitectura KMP realista.
- Priorizar alcance y mantenibilidad.
- Separar la base estructural del desarrollo de logica real.

## Proximos hitos para ensenar en portfolio

- auth real
- backend conectado
- persistencia
- adjuntos
- notificaciones
